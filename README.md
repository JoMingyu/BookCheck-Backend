## 조건
- 하나의 관리자는 하나의 도서관만 관리할 수 있으며 관리할 도서관은 회원가입 시기에 결정된다.
- 관리자에게 주어진 도서관 ID는 바뀌지 않음
- 사용자는 여러 도서관을 사용할 수 있음
